Entity header

The 40 px contextual bar shown when a user opens a specific entity — investment, strategy framework, program, or asset. Sits directly below the global app nav. Three layout zones: left (name, chevron, status, view-mode control), center (4-step workflow tracker), right (collaborators, comments, CTA). Figma node 2676:48493.

Active — Cooper chip

Default state for an active entity with no selected view. Right zone shows the Cooper recommendation chip. Center zone is empty (no workflow tracker).
