Cooper Chat

Cooper is the AI assistant embedded across Copperleaf. It surfaces as a FAB in the bottom-right corner, expands to a floating window or full side panel, and can be triggered inline with a pill input directly in the page. Two input modes: voice (transcribed) and text.

Interactive demo

Click the Cooper FAB or a prompt chip to open the chat. Type a message and press Enter or click Send. Use the expand button in the header to try the side panel view.

Message bubbles

User bubbles sit right-aligned. Inline messages use radius-full; multi-line messages use radius-300 (12px). Hover deepens the fill to --white-200. Prompt chips

Pill-shaped suggestions that appear in the bottom-right zone (beside the FAB) when Cooper has contextual prompts to offer. Also used as follow-up links inside the message thread. Border --border-medium, fill --white-100, radius-full.

Side panel

The expanded view: 450px default width, resizable from the left edge. Full-height, flush to the right side of the viewport with no border-right or border-top. Same background treatment as the floating window. The resize handle is a 1px border with a 3×32px grip pill.

Usage

Cooper Chat exists in three surfaces that share the same visual DNA: the floating window (320px, anchored bottom-right), the full side panel (default 450px, resizable), and the inline pill input (320px, contextual).

All three use the same background treatment: backdrop-filter: blur(var(--blur-strong)) over rgba(0,0,0,0.10) with a Cooper amber radial gradient (bottom-left corner, 20% opacity). The border is always --border-medium (20% white). The pill input inherits the same gradient but scales the radial to fit its 48px height.

The FAB is documented separately on the Buttons page. When the chat is open, the FAB toggles to close the panel. Prompt chips appear to the left of the FAB when Cooper has contextual suggestions from the current page.
