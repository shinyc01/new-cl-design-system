Navigation

The Global Header is the primary application shell. It carries the IFS Copperleaf logo, top-level navigation items, utility actions, and an optional AI insight banner. It renders at full viewport width, fixed to the top of every page.

Usage

The header is a fixed shell component — it sits above all page content at z-index: var(--z-overlay). The backdrop-filter: blur(var(--blur-light)) on --surface-primary-alpha (black/85%) creates the frosted-glass separation from content beneath it.

Navigation items use the element/header-item tokens. Only one item should carry .is-active at a time, reflecting the current section. The active item uses --header-item-fill-active (white/10%); all other states use --white-150 (white/15%) for hover and press.

The AI insight banner (bottomPanel) is shown when Cooper has overnight analysis to surface. The expanded state includes a purple radial gradient bleed from the left edge — this is a Figma-specified art direction value and does not map to a token. The collapsed state (insight/hidden) shows only a pill-shaped expand trigger.
