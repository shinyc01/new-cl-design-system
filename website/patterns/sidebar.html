Sidebar

The Sidebar is a collapsible side navigation panel used inside product views. It groups navigation items into flat parent items and labelled sections with indented child items. A single icon button toggles between expanded (240 px) and collapsed (icon-only) states.

Expanded and collapsed

The collapse button lives in the title row. In expanded mode the panel is 240 px wide and shows all items. In collapsed mode the panel shrinks to the icon button only — use this to reclaim canvas space on dense layouts.

Usage

The Sidebar sits to the left of the main content area. It uses --surface-primary as its background (same as the application shell) and a 1 px --border-base right edge to separate it from the canvas.

Navigation items are 40 px tall with 8 px padding on all sides. Hover and selected states share the same fill — --background-elevation-1-secondary (white/10%) — providing a clear but low-contrast highlight against the dark surface.

Section headings use the --text-disabled colour and Heading/H4/UPPERCASE type style to act as quiet structural labels that don't compete with navigable items. Child items are indented via a 16 px corner-down-right icon and set to --text-secondary by default, upgrading to --text-primary on hover and when selected.

The edit (pencil-plus) button is only revealed when a child item is hovered — it should trigger an inline rename or settings action for that item. It must not appear in the default or selected states.
