Components

Buttons

Verified against the Figma Buttons, Icon Buttons, Prompt button and FAB sheets. Five variants — primary, secondary, tertiary, ghost, accent-cooper (the Cooper AI button, exact gradient layers). Pressed states drop alpha and never shift position.

Variants & sizes

Three sizes: L 40 (pad 12/10, radius 8, Label L) · M 32 (pad 10/8, radius 4, Label M) · S 24 (pad 6/4, radius 4, Label M). Icon↔label spacing 4px; icons 16px (12px on S).

Pressed: fills drop alpha, tertiary border drops to 10%, ghost label dims to secondary. No position shift. There is no destructive/negative button component (the element tokens exist in Figma's variables but no component uses them).

Icon buttons

The full Figma matrix: 2 shapes (square, circle) × 4 variants (primary, secondary, tertiary, ghost) × 3 sizes (L 40 pad 12 · M 32 pad 8 · S 24 pad 4) × 5 states = 120 variants. Icons stay 16px at every size (unlike text buttons). Square radius 8 (L) / 4 (M, S); circles full. One faithful Figma quirk: the ghost square L uses 6px radius. There is no cooper icon button — Cooper is only the FAB.

Notification

Ghost icon button with a danger dot pinned to the bell's top right; the dot carries a surface-colored ring so it reads on any fill.

Prompt chips

Pill-shaped prompt buttons (Cooper AI suggestions). Secondary fill behavior, radius full, sizes L 40 / M 32.

FAB — Cooper button

Inline SVG — no external assets. Resting state has a soft purple/amber glow. Hover triggers scale(1.04) and an aurora keyframe that cycles the glow through purple → pink → amber → indigo over 3.6s. Press scales to 0.97. Disabled drops to opacity: 0.4. Focus uses a drop-shadow ring.

Interactive

Hover, press, tab to it, or toggle disabled.

The React Button primitive in DS/components/core/Button.jsx implements this spec and mounts from the compiled bundle — see Button.prompt.md for the API.
