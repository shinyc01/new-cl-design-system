Quick Stat Card

Compact KPI tile for investment and asset detail overview screens. Shows a metric label, a large value, and an optional description line. Typically used in a 4-up grid.

Default

Title and value only — no description. Suitable when the metric is self-explanatory.

With description

Optional .ifs-quick-stat-card__desc adds a subdued context line below the value — useful for units, basis, or comparison notes.

Usage

Import from DS/components/core/QuickStatCard.jsx. Props: title (string, required), value (string or number, required), desc (string, optional).

<QuickStatCard
  title="Estimated cost"
  value="$4.2M"
  desc="FY2025–2026 · CapEx"
/>

For HTML-only use, apply the classes directly:

<div class="ifs-quick-stat-card">
  <div class="ifs-quick-stat-card__title">Estimated cost</div>
  <div class="ifs-quick-stat-card__value-row">
    <span class="ifs-quick-stat-card__value">$4.2M</span>
  </div>
  <div class="ifs-quick-stat-card__desc">FY2025–2026 · CapEx</div>
</div>

Token notes

One value has no exact DS token: the value's letter-spacing: -0.5px (Figma spec). The nearest tokens are --h1-tracking: -0.6px and --h3-tracking: -0.2px. The raw value is used with an explicit comment in the source. If a --fs-20-tracking token is added to typography.css, update this component.
