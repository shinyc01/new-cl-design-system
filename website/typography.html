Foundations

Typography

One family — Inter — at a compact enterprise scale, 10–24px. All headings are medium (500) with negative tracking; UPPERCASE variants flip to positive tracking. Roboto Mono covers code and tabular contexts. The logo wordmark (Parabole) ships as artwork only.

Scale

Rules

Sentence case for all UI labels, titles and buttons. UPPERCASE is reserved for captions, overlines and Label UPPER variants. Body and Label carry their own letter-spacing — don't drop it. Paragraph spacing for body text is 8px. Never set type below 10px.

Large figures ($478M, 86%) are set in Display — 24/32 medium with −0.6 tracking — not in a heading style.
