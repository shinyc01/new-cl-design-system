IFS Copperleaf

Design system

Tokens, type, color and components for IFS Copperleaf — the decision-analytics suite for asset investment planning. Dark-first, neutral-led, built for dense data surfaces. Everything on this site renders live from the token files, so it always matches the source of truth.

Principles

Neutral-led

Primary actions are white-on-dark; controls are white-alpha. Cooper #C87941 is the brand accent for selected, active and value moments. Purple is the logo and a data-viz hue — never a UI primary.

Dark-first

Black canvas with a four-step surface ladder (#000 → #0F0F0F → #1B1B1B → #373636). Structure comes from hairline borders and tint, not heavy shadows.

Compact & quantified

Inter at 10–24px. Sentence case everywhere, no emoji, outcomes quantified — $478M, +12% vs. baseline, framework coverage 86%.

Getting started

Link the single root stylesheet — it imports every token layer (primitives, palettes, semantic, elements, typography, spacing):

<link rel="stylesheet" href="DS/styles.css">

Then build with the custom properties. Never hard-code hex values:

.card {
  background: var(--surface-2);
  border: 1px solid var(--border-base);
  border-radius: var(--radius-200);
  padding: var(--padding-400);
}
