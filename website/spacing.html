Foundations

Spacing & layout

A 4px grid with half-steps (2, 6, 10) for dense surfaces. Figma models spacing as three roles — paddings inside components, gaps between siblings, margins/gutters for the page grid. Token numbers map to Figma: padding-200 = 8px.

Paddings

Inner space of components and containers.

Gaps

Space between sibling elements (flex/grid gap).

Layout margins & gutters

Page-level grid: margins at the edges, gutters between columns. Both scale by breakpoint.

Usage

.toolbar {
  display: flex;
  gap: var(--gap-200);
  padding: var(--padding-300) var(--padding-400);
}
Layouts are dense and information-rich: fixed left rail + sticky header, scrolling content, generous use of CSS grid/flex with gap.
