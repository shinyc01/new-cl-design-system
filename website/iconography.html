Foundations

Iconography

The real product icon set — 177 glyphs extracted from the Figma Icon Components library. Custom outline style, 1.5px stroke converted to fill, designed for exactly two sizes: 16px (primary) and 12px (compact). Filled variants are used exclusively for status indicators. Icons are monochrome and inherit color via currentColor. Never emoji, never arbitrary sizes, never below 12px.

Usage

The set ships in DS/icons/: individual SVGs (svg/<name>.svg), a symbol sprite (sprite.svg), a registry (icons.json), and icons.js which exposes both for file:// contexts. Inject the sprite once, then reference symbols:

<script src="DS/icons/icons.js"></script>
<script>document.body.insertAdjacentHTML("afterbegin", window.IFSIconSprite);</script>

<svg width="16" height="16" style="color: var(--icon-accent-cooper)">
  <use href="#system-search"/>
</svg>

Rules

Only 12px and 16px — no arbitrary sizes, never below 12px. Monochrome via currentColor or the --icon-* tokens. Filled icons (the filled category) only for status and high-priority accents, primarily on the map. Names are kebab-case, no prefixes beyond their family.

Logo

Primary usage is the plain white lockup on dark. The wordmark is set in Parabole (commercial) and ships as artwork — never recreate it in a webfont.
