Portfolio Bar Chart

Horizontal stacked bar chart showing portfolio distribution across 5 risk/value dimensions (Reliability, Financial, Environment, Social, Governance). Compares You, Peer avg, and Best in class rows. Supports By weight (%) and By budget ($) modes with an interactive switcher.

By weight

Default mode. Segment widths are proportional to percentage values. "You" row uses bright category colors; peer rows use dim variants with tertiary text.

Specs

Container: surface/secondary-alpha background, border/base border, radius-100 (4px), padding 16px top / 20px right-bottom-left. Gap between header and body: 16px.

Row label: 140px fixed width, 12px Medium, text/primary. Segments: 32px tall, gap 2px, border/base border, radius-100. Label text inside segments: 12px 500-weight uppercase, centered, text/primary (bright rows) or text/tertiary (dim rows). Switcher track: black-600 bg (rgba 0,0,0,0.6), 6px radius, 2px padding. Active tab: surface/tertiary-alpha bg, border/base border, 4px radius.

Color tokens — 5 new portfolio palette tokens added to DS/tokens/palettes.css: --portfolio-cat-4-bright (#2699A8), --portfolio-cat-5-bright (#38B5A8), and dim variants for all 5 categories.
