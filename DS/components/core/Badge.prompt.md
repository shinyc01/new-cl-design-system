Compact status / category label — use for plan status (Active/Draft), recommendation counts, and risk flags.

```jsx
<Badge tone="success" dot>Active</Badge>
<Badge tone="copper">1 Recommendation</Badge>
<Badge tone="purple" variant="solid">Optimized</Badge>
```

Tones: neutral · purple · copper · success · danger · warning · info. Variants: soft (default) · solid · outline. Options: `dot`, `square`, `size` (sm/md/lg), `icon`.
